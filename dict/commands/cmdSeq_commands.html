Command: cmdSeq:CmdSequencer

Mnemonic ID Description Arguments
Argument Type Description
CS_RUN 2200 (0x898) Run a command sequence file
fileName Fw::CmdStringArg The name of the sequence file
block SeqBlkState Return command status when complete or not
CS_VALIDATE 2201 (0x899) Validate a command sequence file
fileName Fw::CmdStringArg The name of the sequence file
CS_CANCEL 2202 (0x89a) Cancel a command sequence
CS_START 2203 (0x89b) Start running a command sequence
CS_STEP 2204 (0x89c) Perform one step in a command sequence. Valid only if CmdSequencer is in MANUAL run mode.
CS_AUTO 2205 (0x89d) Set the run mode to AUTO.
CS_MANUAL 2206 (0x89e) Set the run mode to MANUAL.
